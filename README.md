## ⚠️ Pre-requisites

1. [Bun](https://bun.sh) installed.

## 🚀 Quick start

To install dependencies:

```bash
bun install
```

To run the tests:

```bash
bun test
```

To run the tests individually:

```bash
bun test -t Trebuchet
```
